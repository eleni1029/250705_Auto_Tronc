第 13 講 - (語音)非營利組織設計的基本課題 評量題目
友善列印

一、選擇題

1.組織成立的時間愈久規模愈大，而組織規模的大小各有其不同性質的管理問題；組織就像一個生物有機體，其發展過程必然具有興衰更替的生命特性，從這個角度來理解組織設計的問題稱之為：
(  )1. 「組織循環的觀點」
(  )2. 「組織生命週期的觀點」
(  )3. 「組織系統發展的觀點」
(  )4. 「組織適應環境的觀點」

2.組織從嘗試與犯錯的經驗中累積有效的辦事方法訣竅，並加以制度化，這些知識稱之為：
(  )1. 「標準作業程序」
(  )2. 「標準安裝程序」
(  )3. 「彈性作業程序」
(  )4. 「資料隨取程序」

3.強調專業分工、以層級節制為整合機制、權力向上集中、要求標準化的作業，此一組織設計的原則稱之為：
(  )1. 「有機式組織」
(  )2. 「矩陣式組織」
(  )3. 「機械式組織」
(  )4. 「變形蟲組織」

二、問答題

1.請說明組織進行整合的機制為何？
答：
1.安排職位層級較高的人員負責跨部門單位的整合，如機構的執行長或董事。2.鼓勵各部門主管面對面，開誠佈公來溝通。3.指定較資深的主管，賦予其協調跨部門事務的權力。4.就個別專案活動，成立任務小組或專案協調會議。5.打破部門專業分工，改採容納多種專業背景成員的團隊組織結構。6.進行流程再造或組織重組。

2.請討論非營利組織層級結構設計的原則。
答：
非營利組織隨著服務業務的擴大，其組織規模亦相應擴增，而任何大型的組織若欲有效實踐任務，又須藉層級節制的分工體系否則不為功。但非營利組織畢竟不像私人營利事業或公務機關，可以在標準化的作業程序下貫徹「對事不對人」的科層原則，或以例行性的業務運作來達成效率的標準。這是因為，非營利事業的推動大部份仰賴志願性的工作人員，無法單純以嚴密的監督控制予以要求。而社會服務工作所含蓋的專業種類繁多，各專業權威無法形成統屬的關係，指揮命令也無法以層級的單向職權來加以貫徹。社會服務人員一方面被要求對服務對象的需要保持高度敏感性，另一方面又須具備創意活力來設計推動計畫；因此嚴格刻板的官僚體系顯得較不合適。也因此非營利組織的層級設計，比較偏好扁平式的組織和鬆散協調的非正式組織。

3.試說明非營利組織的職位設計原則。
答：
非營利組織的職務工作不像營利企業或公務機關，在固定的分工結構下可以賦予界定明確的職務內容和角色扮演。學者Wolf氏建議，在掌握機構整體工作的質與量後，宜針對以下四群不同性質的人力，妥善分配其職責：
1.支薪的機構全職職員。由於這些人員領受固定薪給，直接構成機構的固定成本，故須課以較嚴格的績效標準與職責。
2.志願工作者。由於這群人不支薪，無由課以嚴格職責；但另一方面又往往是機構最主要的人力來源，所以必須特別注意如何創造誘因來吸引人才、留住人才。
3.獨立契約的受僱者。這些人並非編制內的員工，而是以契約的方式延攬來擔任臨時的專業任務。由於僱佣成本較高，所以特別須要注意工作結果的契約要求。
4.機構外的固定勞務提供者。舉凡機構例行的資料處理、器材維護或衛生安全等，常須以外包的方式，借助機構外的服務公司來提供，較為節省。
